    TURUNAN NUMERIK¶

    A. Definisi Turunan Numerik $$ f'\left( x_{0}\right) = \dfrac{f\left( x_{0}+h\right) -f\left( x_{0}\right) }{h} = \dfrac{f_{1}-f_{0}}{h} $$

    1. Bila persamaan fungsi f(x) diberikan secara eksplisit, maka kita dapat menetukan fungsi turunannya, f'(x),f"(x),.....f(n+1)(x) lalu menggunakannya untuk menghitun nilai turunan fungsi di x = t.f'(x) = 0 lim h ()() h fx +h -fx menggunakannya umtuk menghitiung nilai turunan fungsi di x = t.
    2. Tetapi jika fungsi f(x) tidak diketahui secara eksplisit, tetapi kita hanya memiliki beberapa titik data saja. pada kasus seperti ini kita dapat menemukan nilai turunan fungsi secara analitik.
    3. sebaliknya,pada kasus lain, meskipun f(x) diketahui secara eksplisit tetai bentuknya rumit sehingga menentukan fungsi turunannya merupakan pekerjaan yang tidak mudah.

    B. Persoalan Turunan Numerik

    • Persoalan turunan numerik ialah menentukan hampiran nilai turunan fungsi yang diberikan dalam bentuk tabel

    • Tiga pendekatan dalam menghitung Turunan Numerik yaitu:

    • Hampiran Selisih Maju (forward difference approximation ) $$ f'\left( x_{0}\right) = \dfrac{f\left( x_{0}+h\right) -f\left( x_{0}\right) }{h} = \dfrac{f_{1}-f_{0}}{h} $$

    • Hampiran selisih-mundur (backward difference approximation) $$ f'\left( x_{0}\right) = \dfrac{f\left( x_{0}\right) -f\left( x_{0}-h\right) }{h} = \dfrac{f_{1}-f_{0}}{h} $$

    • Hampiran selisih-pusat (central difference approximation) $$ f'\left( x_{0}\right) = \dfrac{f\left( x_{0}+h\right) -f\left( x_{0}-h\right) }{2h} = \dfrac{f_{1}-f_{-1}}{2h} $$

    Code Program

    def fx(x):
    return float(2*x*x*x)
def fx_y(y):
    return float((y*y*y*y)/2)

batas_bawah = float(input("Masukkan Batas bawah : "))
batas_atas = float(input("Masukkan Batas atas : "))
pembagi = float(input("Tentukan jumlah pembagi : "))

hasil = float((batas_atas-batas_bawah)/pembagi)
x = batas_bawah

print("x\t\tf(x)")

while(x<(batas_atas+hasil)):
    print("{}\t{}".format(x, fx(x)))
    x += hasil

x = batas_bawah+hasil
luas = 0
l0 = fx(batas_bawah)
l1 = fx(batas_atas)


while(x<batas_atas):
    l2 = fx(x)
    luas += l2
    x += hasil

t = (hasil*(l0+l1+(2*luas)))/2
k = fx_y(batas_atas) - fx_y(batas_bawah)
error = float(t-k)

print("Hasil Integral Eksak = {}".format(k))
print("Hasil Integral Trapezoid = {}".format(t))
print("Error = {}".format(error))
